# StackRadar

## Tech stack

- **Framework** — Next.js 16 App Router
- **Language** — TypeScript
- **Styling** — Tailwind CSS v4
- **Code quality** — ESLint, Husky, lint-staged
- **Deployment** — Vercel

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)
